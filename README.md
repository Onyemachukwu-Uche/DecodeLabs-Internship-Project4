# MedCore System Portal

A clean, traditional frontend web interface designed for secure operator form input handling and dynamic input validation. This project focuses on manual core JavaScript DOM logic and direct cascading CSS themes, completely avoiding automated frameworks or bloated libraries to ensure lightweight, predictable browser execution.

## Project Architecture

The application is structured into three standard modules:
* HTML5: High-integrity semantic form inputs with descriptive error indicators.
* CSS3: Class-based manual overrides handling fluid styling transitions and dark background color-swaps without utilizing variable tokens.
* JavaScript: Client-side Input-Process-Output handling that catches form blur events, runs character constraint patterns, toggles visibility, and overrides default form submission behavior to protect memory state.

## Core Features Implemented

* Interactive Mode Swapping: Fully manual light and dark style shifts toggled through a clean user action button.
* Field Masking Toggle: Plain text and dots formatting switch for the key entry input area using structural element updates.
* Live Footprint Validation: Active character scanning on input blur states targeting operator names, formatting strings, and strict alphanumeric symbols.

## Deployment Details

Designed to run statically as a client-side module inside any standardized modern rendering engine.
